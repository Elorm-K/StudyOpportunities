# UK awards reference

Detail for the major UK scholarships `scholarship-matching` considers. Authoritative figures + dates
live in `kb/scholarships/uk.md`; this file is the working catalog with the **gates** spelled out so
the skill can hard-gate before ranking.

> Gate quick-reference for hard-gating. Authoritative values, deadlines, and full country lists live
> in `kb/scholarships/uk.md` (seeded 2026-06-20) — verify the current cycle there before presenting.

| Award | Level | Value | Nationality gate | Field gate | Need/other gate |
|-------|-------|-------|------------------|-----------|-----------------|
| **Chevening** | Master's (taught, 1-yr) | Full (tuition + stipend + travel + visa) | Citizen of a **Chevening-eligible country/territory** | All subjects (career-relevant) | **2,800 hrs work experience**; 3 eligible courses + ≥1 unconditional offer; **return home ≥2 yrs** |
| **Commonwealth Master's** | Master's | Full (tuition + £1,452/£1,781-London + travel) | **Low/middle-income Commonwealth** (44-country list) | All (dev-theme/SDG aligned) | ≥**2:1**; **unable to afford**; **via Nominating Agency** (no direct apply) |
| **Commonwealth PhD** | PhD (36 mo) | Full | **LDC/vulnerable-states** stream (19-country list, incl. Nigeria/Pakistan/Kenya) — distinct from high-income stream | All (dev-theme/SDG) | ≥**2:1**; unable to afford; via Nominating Agency |
| **Gates Cambridge** | PG at Cambridge | Full-cost (tuition + £22,050/yr + travel + visa) | **Any country outside the UK** | Any Cambridge subject | Cambridge admission required; no return-home requirement |
| **Clarendon** | PG at Oxford | Full (fees + ≥£20,780/yr living) | **All nationalities — none** | No restriction | Oxford admission + outstanding merit; **auto-considered** (no separate application) |
| **GREAT** | Master's (taught) | **Partial: ≥£10,000** tuition | **18 specific countries** (2026-27 list) | Varies by university/strand | Enrol at participating university; **apply direct** (no central deadline) |

## Notes

- Cambridge/Oxford college awards (Gates, Clarendon) require admission to the university first —
  treat the university shortlist and these awards together.
- Verify each award's current cycle dates and eligibility via `knowledge-base-update` before
  presenting (figures shift each cycle).
