# US required documents by level

> **Deferred (background priority).** Conventional baseline for `document-checklist`; the exact list
> is confirmed **on-demand by `knowledge-base-update`** against the target program's page (and Common
> App) per client. Leading countries (Bangladesh, Ghana, Nigeria) are prioritized for verification.

## Undergraduate (Common App)
- Common App essay + per-school supplemental essays
- Counselor + teacher recommendations
- Official transcript / school report
- Standardized tests (SAT/ACT) where required (many test-optional — verify per school)
- English language test (TOEFL/IELTS/Duolingo) for non-native speakers
- CSS Profile / financial-aid forms (internationals seeking aid)

## Graduate (direct to program)
- Statement of Purpose (SOP)
- 3 letters of recommendation (typical)
- Official transcripts
- GRE/GMAT where required (many waived — verify per program)
- English language test
- Writing sample / portfolio (field-dependent)
- CV/résumé
